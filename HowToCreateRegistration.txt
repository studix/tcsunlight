Neues Turnier anlegen:
1. Unter http://www.tcsunlight.ch/tournaments/new Turnier anlegen
2. Mit putty auf web server einloggen. 
3. Spielplanverfügbarkeitsdatum ändern:
	mysql -u studer_root -p 
	use studer_sunlight;
	select * from tournaments;
	update tournaments set [spaltenname] where id = [id];

Neue Anmeldung erstellen
http://www.tcsunlight.ch/registrations/new?tournament_id=5

Alle Anmeldungen sind verfügbar unter
http://www.tcsunlight.ch/registrations?tournament_id=5
